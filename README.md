A sleek Pomodoro focus timer built on top of the Intercom stack by Trac Systems.

## Features
- 3 modes: Focus (25min), Short Break (5min), Long Break (15min)
- Circular progress ring that drains in real time
- Task input to track what you're working on
- Auto-switches between focus and break sessions
- Tracks total sessions, focused minutes, and streak
- Desktop notifications when a session ends

## How to Run
1. Clone or download this repo
2. Open `pomodoro-timer.html` in any browser
3. No installation or server needed
